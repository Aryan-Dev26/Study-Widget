# Changelog

All notable changes to Study Timer Pro will be documented in this file.

## [1.0.0] - 2024-12-09

### Added
- Initial release of Study Timer Pro
- Countdown timer with custom durations
- Live world clock with 30+ timezones
- Pomodoro technique integration
- Quick preset buttons (15min, 25min, 30min, 1h, 2h, 5h)
- Session statistics tracking (daily, weekly, monthly)
- Streak counter for daily study habits
- Gamification system with XP and levels
- 6 beautiful themes (Dark, Light, Blue, Green, Purple, Nord)
- Sound alerts and desktop notifications
- Task tracking and labeling
- CSV export for study data
- Keyboard shortcuts (Space, R, Esc)
- Draggable widget with position memory
- Opacity control (50-100%)
- Auto-save configuration
- Always-on-top mode
- Minimal, distraction-free UI

### Features
- Pause/Resume functionality
- Reset timer option
- Auto-minimize on start
- Color-coded time warnings
- Session history logging
- Daily goal tracking
- Settings persistence
- Multi-tab interface

### Technical
- Built with Python and Tkinter
- JSON-based data storage
- Windows 10/11 toast notifications
- Modular code architecture
- Configuration management system
- Statistics tracking system
